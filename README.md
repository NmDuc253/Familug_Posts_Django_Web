## The simple static web made by Django
**1. Create virtualenv**
``` shell
$ python3 -m venv venv 
```
**2. Active virtualenv**
``` shell
$ source venv/bin/active
```
**3. Install packages**
``` shell
(venv) $ python3 pip install -r requirements.txt
```
**4. Run app**
``` shell
(venv) $ python3 manage.py runserver 0.0.0.0:8000
```
